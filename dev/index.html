Open position
Software developer
for LightNet

MediaWorks is looking for a software developer to create software that enables ministries to share media with each other in a decentralised way and quickly create interactive community websites and webapps.

We are looking for a Software developer to strengthen our experienced development team. You will be working in a team sharing the vision to create software that enables ministries to thrive digitally and share God's love with the least reached in new ways.

LightNet is an online product that enables ministries to run their own media distribution platforms. This digital tool helps ministries to publish curated content in the language and context of their focus communities.

Technologies

We are using the following stack:

  • HTML, CSS, JavaScript, Progressive web apps (PWA)
  • Resource Description Framework (RDF) database
  • Docker, Linux
  • Continuous integration and delivery pipeline
  • Automated testing tools
  • Cloud Infrastructure and Infrastructure as Code tools

What you would be working on

We are building a highly decentralised system that has a couple of components. Depending on your interests and skills, you can work with the primary focus on the frontend, backend or infrastructure aspect. We are not only building a content management system (CMS) but also a generic frontend that is highly configurable by the ministries using it. For example: library.mediaworks.global and nadija.org.

Some ministries will need a fully customised frontend. We are building the system in an agile way and will adapt our focus and direction depending on the use cases of the local ministries.

Requirements

  • Understanding of and compliance with OM core values and statement of faith
  • A few years of work experience in the industry
  • Familiarity with some of the above-mentioned technologies
  • Interest in Open Source and decentralisation
  • Team player
  • Willing to learn, adapt and work in an international hybrid team
  • Good communication skills, able to communicate in English

Location

It is possible to work in Stockerau (Austria) or remotely from anywhere in the world with good Internet connection.

Financial

This is a non-paid position. All our workers need to raise financial support. It is also possible to work with us as a volunteer.
